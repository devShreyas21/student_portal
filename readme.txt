# Backend Setup Instructions

Follow these simple steps to run the backend locally:

## Step 1: Clone the Repository
1. git clone https://github.com/devShreyas21/student_portal.git
2. cd student_portal


## Step 2: Create Environment File
Create a `.env` file in the root directory with the content send to neha.k@nextastra.com  


## Step 3: Install Dependencies and Run
1. npm install
2. node server.js

The backend should now run at http://localhost:5000/
